# Changelog

## v1.2.0 (2026-07-13)

### 🆕 新功能

- 新增全屏模式，鼠标移至顶部唤起独立控制条，可快速切换编辑 / 预览 / Diff 视图，并通过“标签 / 文件”或 `Ctrl+B` 打开全屏文件侧边栏。全屏模式下 AI 助手改为小窗模式，拖动顶部调节位置，点击外部关闭弹窗，实现即用即走，有助于提升专注度。
- 新增浅色模式，白色主题。现主题模式改为 2 套：暖色 + 深色以及浅色 + 深色，可到设置 - 通用 - 明亮配色切换套装，`Ctrl+Shift+L` 以及软件右上角主题快捷键实现套装内主题快速切换。
- 新增 AI 吉祥物头像选项，可在设置页的 AI 头像入口中启用，AI 生成回复时会有“思考 - 写作”小动画。
- 新增模式闲时预热设置，可在设置页中开启，用于提升编辑 / 预览 / Diff 等模式切换体验。

### 🐛 修复

- 修复 Markdown PDF 导出时本地图片无法读取的问题。
- 修复 Markdown PDF 导出时普通段落换行丢失的问题。

### ⚡ 优化

- 优化界面视觉降噪，提升视觉专注度。

## v1.1.0 (2026-07-09)

### 🆕 新功能

- 新增全屏专注模式，支持独立全屏控制栏、侧边栏 / AI 面板快速开关和编辑视图切换，适合沉浸式写作、阅读与对照。

### 🐛 修复

- 修复长文本流式输出稳定性问题，降低回答中断和增量内容丢失风险。
- 修复知识库参考来源无法打开的问题。
- 修复深色模式下 Mermaid 图表与设置输入文字可读性不足的问题。
- 修复侧边栏长文件名挤压布局的问题，并统一悬浮提示样式。

### ⚡ 优化

- 优化同步文件路径去重与 Diff 显示，减少同一文件被重复识别时的干扰。
- 移除顶栏当前文档名显示，为窗口拖拽和全屏专注模式保留更干净的顶部区域。

## v1.0.5 (2026-07-06)

### 🆕 新功能

- 新增用户使用说明书，含 38 张功能配图（`docs/USER_GUIDE.md`）
- 文件标签右键菜单新增"打开文件位置"选项，调用 Windows 资源管理器定位并选中文件
- 新增 Tauri 桌面命令 `reveal_file_in_folder`
- 粘贴/拖入 Markdown 的图片资源名限制为 ASCII 安全字符，避免中文文档名生成含有中文的 assets 文件名

### 🐛 修复

- 修复 AI 历史聊天问答关联与分页错位：增加 `parent_id` 显式父子关联，按完整 turn 分页，补充 5 组回归检查
- 修复编辑模式目录高亮：基于可视行实现标题跟踪，解决切换面板后的 heading 同步问题
- 修复 PDF/HTML 导出：本地图片嵌入 data URL、等待字体和 Mermaid 渲染完成后再打印、超宽代码/表格自动换行
- 修复中文图片路径导致预览加载失败，兼容已存在的含中文图片资源
- 修复 Markdown 图片插入与本地预览授权

### 🔧 构建

- 更新 CLAUDE.md 发布流程文档
